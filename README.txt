git-study 폴더를 깃 저장소로 만들겠다.
깃 저장소인 폴더는 .git 폴더를 가지고 있다.

.git 폴더를 가지는 바로 상위의 폴더는 하나의 폴더가 됩니다. 그 이하에 생성되는 폴더들은 소위 '작업물'들이 됩니다.
내가 생각했을 때, 어떻게 각 폴더들을 정리하는지 궁금했었는데, .git 폴더의 위치로 확인합니다.

.git 폴더가 있는 현재폴더를 특별히 '워킹폴더'라고 부릅니다.

혼동하는 개념 :
.git 폴더를 포함하는 현재폴더를 [워킹트리]라고 합니다. 
.git 폴더를 [로컬저장소]라고 부릅니다. 혼동하기 쉽습니다.
Github에 업로드를 하는 상황을 생각합시다. github가 [원격저장소]가 됩니다. 그러나 혼동하지 말아야할 것은, github 즉 원격저장소에 업로드하는 것은 워킹트리가 아닌 로컬저장소입니다.
[git 저장소] 엄밀하게는 로컬저장소를 일컫습니다. 그러나 넓게는 작업폴더를 가리키는 말이기도 합니다.
